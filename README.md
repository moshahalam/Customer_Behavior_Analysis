# 📊 Customer Shopping Behavior Analysis

## 📌 Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The goal is to uncover insights into spending patterns, customer segmentation, product preferences, and subscription behavior to support strategic business decisions.

The project demonstrates a complete end-to-end data analytics workflow — from data cleaning and exploratory analysis to SQL querying, dashboard creation, and business reporting.

---

## 📂 Dataset
- **Type:** Transactional customer purchase data  
- **Records:** 3,900 purchases  
- **Key Attributes:**
  - Customer demographics (Age, Gender)
  - Product details (Category, SKU, Ratings)
  - Purchase metrics (Total Price, Discount Applied)
  - Subscription status
  - Shipping type

The dataset was cleaned and transformed before analysis.

---

## 🛠 Tools & Technologies
- **Python (Pandas, NumPy, Matplotlib, Seaborn)** – Data loading, cleaning, and EDA  
- **SQL (PostgreSQL / MySQL / SQL Server)** – Data querying and aggregations  
- **Power BI** – Interactive dashboard development  
- **Gamma** – Presentation creation  
- **Jupyter Notebook** – Analysis workflow environment  

---

## 🔎 Project Workflow

### 1️⃣ Data Loading & Cleaning (Python)
- Imported dataset using Pandas  
- Handled missing values  
- Converted data types  
- Removed duplicates  
- Created derived columns  
- Prepared structured dataset for SQL & BI tools  

### 2️⃣ Exploratory Data Analysis (EDA)
- Revenue distribution analysis  
- Customer age segmentation  
- Discount usage trends  
- Product category performance  
- Subscription behavior patterns  

### 3️⃣ SQL Analysis
Performed analytical queries including:
- Revenue by product category  
- Discount impact evaluation  
- Customer segmentation analysis  
- Identifying unsold products  
- Ranking top-performing items  
- Average spending comparisons  

### 4️⃣ Power BI Dashboard
Developed an interactive dashboard featuring:
- Total Revenue KPI  
- Average Spending per Customer  
- Revenue by Category  
- Subscription vs Non-Subscription Comparison  
- Age Group Revenue Analysis  
- Shipping Preference Insights  
- Top-Rated & Best-Selling Products  

---

## 📊 Dashboard Highlights
- Interactive slicers and filters  
- KPI cards for performance tracking  
- Visual comparison across customer segments  
- Dynamic revenue analysis  
- Business-focused insights for decision-making  

---

## 📈 Key Insights
- Identified high-revenue age groups  
- Determined top-performing product categories  
- Observed the relationship between discounts and sales volume  
- Analyzed subscription impact on overall revenue  
- Evaluated shipping preferences among customers  

These insights can support marketing strategies, pricing optimization, and customer retention initiatives.
